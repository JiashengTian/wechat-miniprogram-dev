## 一、快速开始

1. 在微信中打开小程序（无需注册登录，数据保存在本机）。
2. 点击首页底部的 **「+ 添加习惯」**，输入一个习惯名称（如「早起」「读书」），点击 **「记下」**。
3. 在习惯列表中，点击习惯左侧的 **圆圈** 即可完成今天的打卡。
4. 坚持每天打卡，连续天数和火花值会随之增长，解锁成就。

> 💡 所有数据默认存储在手机本地。卸载小程序或更换设备前，请注意数据可能无法恢复，建议提前截图留存。

## 三、添加习惯

1. 点击首页底部 **「+ 添加习惯」**。
2. 在输入框中填写习惯名称（最多 20 字）。
3. 可点击下方的 **建议词**（如系统推荐的常见习惯）快速填充。
4. 点击 **「记下」** 完成添加，新习惯会出现在列表顶部。

> 添加后，可点击习惯卡片进入详情，补充难度、周目标、分类、备注等信息。

## 四、每日打卡

- **完成打卡**：点击习惯左侧的圆形勾选框。勾选后变为蓝色实心圆 + 白色对勾。
- **取消打卡**：再次点击同一圆圈，即可取消当天的打卡（连续天数会相应重置计算）。
- **全勤提示**：当今天所有习惯都已打卡时，会弹出「诸事皆毕 🎉」鼓励弹窗。

> 打卡只对「当天」生效。若想补之前的日期，请使用 [补打卡](#十四补打卡) 功能。

## 五、习惯卡片详解

每个习惯在列表中以一张卡片呈现：

```
[✓] [习惯名]
     [⭐⭐] [分类] [🎯周3✓]      [连续5天🔥]
                          📊   🏷️   🗑
```

| 元素 | 说明 |
| --- | --- |
| ✓ 圆圈 | 今日打卡按钮，点击切换打卡状态 |
| 习惯名 | 轻点一次显示全名（Toast）；**双击**或**长按**进入重命名编辑 |
| ⭐ 难度星 | 设置的难度星级（1–5 星），用于区分习惯重要程度 |
| 分类标签 | 所属分类（如「健康」「学习」），可空 |
| 🎯周目标 | 形如「周3✓」，表示本周目标 3 次，✓ 表示本周已达标 |
| 🔥 连续天数 | 当前连续打卡天数 |
| 📊 | 打开习惯详情 |
| 🏷️ | 快速修改该习惯的分类 |
| 🗑 | 删除该习惯（有二次确认） |

## 六、习惯详情与设置

点击习惯卡片上的 **📊** 或卡片中部，打开详情弹窗：

- **连续统计**：显示当前连续天数与累计打卡次数。
- **难度**：点击 ⭐ 设置 1–5 星难度。
- **🎯 周目标**：通过选择器设定「每周打卡几次」（可选 0 / 3 / 5 / 7）。设定后卡片上会显示「周 N」标签，并在达标时打勾。
- **备注**：可填写该习惯的个性化说明，点击「保存备注」生效。
- **删除习惯**：底部红色文字，点击后二次确认删除（删除后该习惯的打卡记录一并清除）。

### 重命名习惯
- **轻点**习惯名：弹出 Toast 显示完整名称（适合名称被截断时查看）。
- **双击**（350ms 内连点两次）或 **长按**：进入编辑模式，修改后保存即可。

## 七、分类管理

分类用于给习惯分组，方便筛选与统计。

**方式一：从首页分类条**
- 点击分类条上的「全部」或某个分类名，可筛选显示对应习惯。
- 点击「+分类」可新建分类。

**方式二：从习惯卡片**
- 点击卡片上的 **🏷️** 按钮，弹出分类选择框，列出所有分类 +「无分类」；点击即设定，也可在此进入「管理分类」。

**方式三：从设置**
- 打开 ⚙ 设置 → 分类 → 管理分类，可统一增删分类。

## 八、周目标

周目标帮助你为习惯设定「每周至少打卡几次」的小目标。

1. 在习惯详情中，点击 **🎯 周目标** 右侧的选择器。
2. 选择目标次数：无 / 每周 3 次 / 每周 5 次 / 每周 7 次。
3. 保存后：
   - 卡片标签显示「周 N」；
   - 本周内打卡达到目标次数后，标签后出现「✓」表示达标；
   - 在看板与周报中可查看目标完成情况。

> 周目标按自然周（周一至周日）统计。

## 九、火花值与排序

**火花值**是本小程序的核心激励指标，反映一个习惯的「坚持热度」。

```
火花值 = 累计打卡次数 + 连续天数 × 2
```

- 打卡次数越多、连续越久，火花值越高。
- 习惯列表会 **按火花值从高到低自动排序**，火花值高的习惯排在前面。
- 排序变动时，列表会以顺滑动画平滑过渡，其他卡片也会相应位移。

> 火花值仅用于排序与激励展示，不影响任何功能可用性。

## 十、成就系统

坚持打卡可解锁成就，共 **20 项**。

- 打开方式：首页 📊 看板 → 滚动到「成就（20项）」区域。
- 达成条件示例：连续打卡 7 天、累计打卡 30 次、单周全勤等（具体以小程序内展示为准）。
- 解锁时会有成就弹窗提示。
- 在看板顶部可看到「成就：N」总数。

## 十一、番茄钟专注

番茄钟帮助你为习惯配套专注时间。

1. 点击首页右上角 **🍅** 打开专注计时。
2. **选择时长**：
   - 点击预设按钮（如 25 分 / 45 分 / 60 分）；
   - 或在输入框填写自定义分钟数，点击「设定」。
3. **控制**：
   - 「开始 / 暂停」按钮控制计时；
   - 「⏭ 跳过」跳过当前休息阶段；
   - 「⟲ 重置」归零重新计时。
4. 计时采用翻牌动画显示分:秒。
5. 底部显示「今日番茄：N ｜ 总番茄：N」，数据在看板中汇总。

> 可在 ⚙ 设置中开启 / 关闭「番茄钟振动」。

## 十二、数据看板

点击首页标题「习惯」或右上角 📊，打开数据看板：

- **总番茄**：累计完成的番茄钟数量。
- **成就**：已解锁成就数量。
- **火花排行**：各习惯按火花值排名，一目了然你的坚持重心。
- **成就列表**：20 项成就及解锁状态。

## 十三、周报与月报

点击首页「周报」链接打开统计报告：

- **全勤**：本周/本月每天都有打卡的天数。
- **零打卡**：完全没有打卡的天数。
- **各习惯**：每个习惯的完成次数 / 总天数及完成率（%）。
- **最佳日**：完成习惯最多的那一天。
- 点击底部按钮可在 **周报 / 月报** 之间切换。

---

## 十四、补打卡

若某天忘记打卡，可补录历史日期：

1. 点击首页「补打卡」链接。
2. 选择要补打卡的 **习惯**。
3. 在日期列表中选择目标日期（显示已打卡 / 未打卡状态）。
4. 点击对应日期的「补」按钮完成补卡。

> 补打卡会正常计入连续天数、火花值与周目标统计。

## 十五、日历视图

点击首页「日历」链接打开月历：

- 以月为单位展示每天的打卡密度。
- 顶部「‹ / ›」可切换上一个月 / 下一个月。
- 点击某一天可查看当天的打卡详情提示。

## 十七、常见问题（FAQ）

**Q1：数据会同步到云端吗？**
A：本小程序为纯本地工具，数据仅保存在当前手机。更换设备或卸载前请自行备份（如截图）。

**Q2：不小心取消了打卡，连续天数为何变了？**
A：连续天数基于最近一次未中断的打卡计算，取消某天打卡会影响连续统计，重新打卡即可恢复累计。

**Q3：习惯排的顺序为什么会变？**
A：列表按火花值（累计打卡 + 连续天数×2）自动排序，坚持越久越靠前，排序变化带有平滑动画。

**Q4：如何修改习惯名称？**
A：在列表中**双击**习惯名（或长按）进入编辑；轻点仅显示全名。

**Q5：周目标没达标会有惩罚吗？**
A：不会有任何惩罚，周目标仅为自我激励，未达标时标签不显示 ✓。

**Q6：误删了习惯能恢复吗？**
A：删除为永久操作，无法通过小程序恢复，请谨慎使用删除功能。

**Q7：如何彻底清空数据重新开始？**
A：⚙ 设置 → 重置全部，确认后清空所有本地数据。
