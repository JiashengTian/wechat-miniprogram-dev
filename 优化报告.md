## 📋 技术细节

### 动画关键帧
```css
@keyframes fadeIn - 渐入动画
@keyframes fadeInDown - 向下渐入动画
@keyframes slideUp - 向上滑动动画
@keyframes slideInRight - 向右滑动动画
@keyframes popIn - 弹出动画
@keyframes popCheck - 复选框弹跳动画
@keyframes shimmer - 微光动画
```

### 性能优化
- 使用 will-change 提示浏览器优化
- 使用 translateZ(0) 启用 GPU 加速
- 优化 setData 调用频率
- 压缩 CSS 选择器复杂度

### 分类管理
- 独立弹窗，不在设置里
- 快速添加/删除分类
- 删除前安全检查
- 颜色示例展示
